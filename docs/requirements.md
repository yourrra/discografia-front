# requirements.md

## Scope v0 (MVP)

Проект состоит из двух разделов:

### 1) Terminal UI

Пользователь видит интерфейс терминала.
Требования:

- Ввод команд (строка), отправка Enter
- История команд (стрелки вверх/вниз)
- Вывод результатов команд в виде "строк терминала"
- Поддержка "виртуальных" команд (пока без реального shell), например:
  - help
  - open <pageId>
  - list
  - clear
- Автоскролл вниз при новом выводе
- Базовая стилизация под терминал (моноширинный шрифт, подсветка)

Опционально:

- Автокомплит команд (Tab)
- Подсветка ошибок и статусов

### 2) Canvas UI (как Miro-полотно)

Полотно бесконечное/большое, на нём узлы (nodes), где узел = страница.
Требования:

- Панорамирование и zoom (мышь/тачпад)
- Узлы отображаются карточками
- Узел содержит заголовок и список ссылок на другие узлы
- По клику на ссылку: плавно переместиться/подъехать камерой к целевому узлу
- Возможность анимированного появления новых узлов при клике (если узла не было)

Опционально:

- Миникарта, поиск по узлам, "fit view"
- Сохранение положения/масштаба (localStorage)

## Non-functional

- 60 fps при пане/зуме на средних графах (100-300 узлов)
- Предсказуемое состояние: без гонок и скрытых сайд-эффектов
- Линтеры и форматирование — зелёные в CI
- Доступность: элементы кликабельны с клавиатуры, есть focus states
